# Scripts Season 1 (2023–2025)

This repository contains my earliest programming work, created between 2023 and 2025.

## About

This is where my programming journey began.
Most of the files here were made while I was still learning the basics, experimenting with ideas, and trying out different things without much structure.

You’ll find a mix of:

* Random HTML experiments
* Early Python scripts
* Small “vibe-coded” projects

At this stage, I wasn’t focused on writing clean or optimized code — I was focused on exploring and learning.

## Why the Code is Messy

The code in this repository may appear unorganized or inconsistent.
This is intentional — it reflects my starting point as a programmer, when I was still figuring things out and didn’t fully understand best practices or structure.

Rather than cleaning it up, I chose to keep it as-is to accurately represent how I began.

## Purpose

This repository is kept as a record of my starting point as a programmer.
It shows how I began, and serves as a reference for how much I improve in later projects.

This is **Season 1** of my programming journey — future seasons will reflect my growth, better structure, and more advanced projects.

## Note on Changes

A small number of files have been slightly modified before uploading to remove personal information (such as school name) for privacy reasons.
Apart from these minor changes, the code remains exactly as it was originally written.

## Final Note

This may not be perfect code, but it represents the beginning of my journey.
Future repositories will show how far I’ve come!
