# Japanese Learning Trainer - 日本語 Trainer

Learn Japanese from absolute zero with cute anime mascot teachers! This interactive web app teaches Hiragana, Katakana, and basic Kanji through visual lessons and quizzes.

## Features

- **Learn Hiragana** - Master the basic Japanese alphabet with Fox Sensei 🦊
- **Learn Katakana** - Discover foreign words with Neon Gamer Cat 🐱
- **Learn Basic Kanji** - Explore Chinese characters with Samurai Raccoon 🦝
- **Quiz Zone** - Test your knowledge with 15-question quizzes
- **Progress Tracking** - Your progress is saved in your browser
- **Audio Pronunciation** - Hear each character pronounced correctly
- **Beautiful Neon Design** - Tokyo-night aesthetic with anime mascots

## How to Use

1. Start with Hiragana if you're a complete beginner
2. Each lesson teaches characters step-by-step with examples
3. Complete each section to unlock the quiz
4. Take quizzes to test your knowledge and get feedback
5. Review characters you got wrong
